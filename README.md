# Japan Trip Interactive Guide

Mobile-first travel app for a 10-day Japan itinerary with interactive map and food guide.

## Tech Stack

- **Frontend:** React 19 + Vite 7.3
- **Routing:** React Router v7 (data mode)
- **Hosting:** Cloudflare Pages
- **Build:** Optimized for sub-3-second Time to Interactive (62KB gzipped initial bundle)

```bash
npm install

```

Visit `http://localhost:5173` to see the app running locally.

```bash
npm run build
```
